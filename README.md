### Clone the repository

git clone https://github.com/Rinwaoluwa/shippex
cd shipppex

### Install dependencies 

npm install
# or
yarn install

### Running the App

Start the development server:

```bash:README.md
npx expo start
# or
npm run start
# or
yarn start
```

### Building for Production

To create a production build:

```bash:README.md
npx expo build:ios
# or
npx expo build:android
```
